This is a personal birthday website project created with React + Vite.

## Features
- **Hero Section**: Animated greeting with confetti explosion.
- **Memory Gallery**: A responsive photo grid with lightbox view.
- **Music Player**: Background music toggle.
- **Premium Design**: Glassmorphism, gradients, and smooth animations using `framer-motion`.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm run dev
    ```

## Customization

### 1. Changing Photos
Open `src/components/Gallery.jsx` and update the `images` array with your own photo URLs. You can put images in the `public` folder and reference them like `/my-photo.jpg`.

### 2. Changing Music
Open `src/App.jsx` and update the `<source>` `src` attribute in the `<audio>` tag with your preferred song URL.

### 3. Changing Text
Open `src/components/Hero.jsx` to change the greeting message.

## Applied Technologies
- React
- Vite
- Framer Motion (Animations)
- Canvas Confetti (Effects)
- Lucide React (Icons)
- Vanilla CSS (Styling)
